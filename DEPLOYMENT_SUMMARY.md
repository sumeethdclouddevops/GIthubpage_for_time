## 🔧 Customization Examples

### Add More Agents
Edit `handover-generator.html`:
```html
<option value="Agent11">Agent 11</option>
<option value="Agent12">Agent 12</option>
```

### Change Shift Times
Edit shift radio buttons:
```html
<input type="radio" name="shift" value="Your Time Range">
```

### Add Data Center
Edit `index.html` dataCenters array:
```javascript
{ 
    id: 'new-dc', 
    name: 'New DC (CODE)', 
    zone: 'Timezone/City',
    ring: 1 or 2
}
```
